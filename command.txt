How to use it:
Make it executable:
Run this in your terminal once:
chmod +x start_sim.sh

Run it:
./start_sim.sh (Headless)
./start_sim.sh --map (sitl map version)
./start_sim.sh --qgc (qgc version no map)

2. Advanced Testing Modes

You can append flags to customize the session:

With Stress Test: 

./start_sim.sh --stress (Triggers the CPU spike 100s after start).

./start_sim.sh --map --stress (it would run sitl plus map and stress test for it as well)

./start_sim.sh --qgc --stress (it will run qgc sitl communication and stress test for it )

Full Suite: ./start_sim.sh --fullmode (Enables Map, QGC, and the Stress Test).
